C++ Primer读书笔记(二) 字符串、向量和数组

Posted on 2020-06-16 | In reading note |

3.1命名空间的using声明

目前我们使用的函数基本属于命名空间std，如果每次调用都用std::则比较繁琐，因此我们可以在程序开头声明:
using namespacce std;
头文件不应该包含using！否则会造成一些始料未及的错误。

3.2标准库类型string

3.2.1初始化

1.拷贝初始化： string s1 = "hello";
2.直接初始化： string s2("hello");

3.2.2操作

1.从字符串is中读取一行赋值给s： getline(is,s);
2.判断s是否为空：s.emtpy();
3.获得s的大小: s.size();
4.连接s1和s2: s1+s2;

3.3标准库类型vector

3.3.1初始化

vector是模板并非类，必须进行实例化。
vector < int> vi;

3.3.2vector操作

push_back(val);
表示向容器push入一个对象val至尾端。 v.empty(); v.size(); 类似string.

3.4迭代器

iterator是用于迭代的一个容器，有begin和end成员，用auto类型表示，用于表示首位置和最后位置。通常操作符： = = 和 ! = 进行逻辑判断 *iter返回iter所指元素的引用 iter->mem，相当于(\*iter).mem

3.5数组

熟悉的数组,初始化的维度必须是常量。数组不允许拷贝和赋值，即不允许使用一个数组初始化另一个数组。 数组的第一个元素,nums[0]的地址即nums的值，即string *p2 = nums[0]相当于p2 = nums。
